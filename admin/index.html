上线清单

  1. 发布云托管后打开 /admin/，在「运行管控」填写：呆呆 AI / 呆呆 Image 密钥、小程序对接域名
  2. 小程序 app.js 的 apiBase 填同一域名，并加入 request 合法域名
  3. 云托管仍需 ADMIN_PASSWORD；小程序登录需 WECHAT_APPID / WECHAT_SECRET
  4. 网页端与后台同源已对接（相对路径 /api/*）；用户聊天请用小程序
  5. 对外永不展示供应商名，一律「呆呆 AI / 呆呆 Image」
  6. 维护模式会拦截网站页面与全部业务 API；管理后台仍可进。多副本时设置写在容器本地，建议最小副本先设为 1
